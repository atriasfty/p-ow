# How to Create a Server

Follow these steps to deploy your very own Project Overwatch (POW) command center for your ER:LC / PRC community.

### 1. Account Setup
1. Go to **[pow.ciankelly.xyz](https://pow.ciankelly.xyz)** and click **Sign In**.
2. Sign in with **Discord**. This is required because POW uses your Discord rank to determine your permissions.
3. Once logged in, click your profile icon and ensure your **Roblox Account** is also linked.

### 2. Initial Deployment (Step 1)
1. On your dashboard home page, click the green **Plus (+)** button or the **Create Server** button.
2. **PRC API Key:** Enter your PRC Server API Key. (Find this in your ERLC Server Settings -> API tab).
3. **Discord Guild:** Select the Discord server you want to link. 
    - *Note:* You must have **Administrator** or **Manage Server** permissions in the Discord guild for it to show up here.
    - *Note:* The **POW Discord Bot** must be invited to your server. Use the link provided in the wizard if it's not already there.
4. Click **Verify & Continue**.

### 3. Basic Configuration (Step 2)
1. **Dashboard Admin Role:** Select the Discord role that should have full "Owner" access to this dashboard (usually your "Owner" or "High Command" role).
2. **General Staff Role:** Select your primary staff role in Discord. This role will automatically grant "Viewer" access to the Mod Panel for your staff.
3. **System Logs Channel:** Select the Discord channel where you want POW to send moderation logs, staff requests, and system alerts.

### 4. Finish Deployment
1. Click **Deploy & Finish**.
2. POW will set up your database, link your roles, and redirect you to your new **Admin Dashboard**.

> [!TIP] Need to Change Settings?
> You can always update your API Key, staff roles, and log channels later by going to **Admin Panel > Settings** on your server's dashboard.
