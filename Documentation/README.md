# Welcome to Project Overwatch

Project Overwatch (POW) is a comprehensive moderation and staff management platform built specifically for ER:LC / PRC servers on Roblox. It bridges the gap between your game server, your Discord server, and your staff team.

{% hint style="info" %}
The official dashboard is located at: [**pow.ciankelly.xyz**](https://pow.ciankelly.xyz)
{% endhint %}

### Key Features

* **Real-time Logging:** Synchronize in-game logs directly to Discord and the POW Dashboard.
* **Staff Management:** Track shifts, enforce quotas, and manage Leave of Absence (LOA) requests.
* **Automation Engine:** Trigger Discord notifications and game commands based on server events.
* **Form Builder:** Create recruitment applications and surveys with automatic role granting.
* **Pro Features:** Protect your server with automated Raid Detection and AI-powered player risk assessments.

Explore the sidebar to get started with setting up your community or learning more about specific features.
